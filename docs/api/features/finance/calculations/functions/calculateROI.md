# Function: calculateROI()

> **calculateROI**(`profit`, `totalInvestment`): `number`

Defined in: features/finance/calculations.ts:41

Calculates the Return on Investment (ROI) percentage.
Formula: (Net Profit / Total Investment) \* 100

## Parameters

### profit

`number`

The net profit (or loss)

### totalInvestment

`number`

The total amount invested (all costs)

## Returns

`number`

ROI percentage (e.g., 25.5 for 25.5%)
